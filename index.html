政策轉型與產品定位

One-Stop Service

核心政策變更

衛福部公告之「救急救難一站通」計畫，針對四大重症之交換機制已由 EEC 線上交換 轉為 XML 實體檔案匯出。

產品因應目標

  • 多開發排程執行報告匯出 (一站通轉檔模組)
  • 產出符合 EEC 規格之 XML
  • 提供 EMR 1.0 / 2.0 雙軌解決方案

OHCA

院外心跳停止

Trauma

重大外傷

腦中風

急性心肌梗塞

重點備註： 報告檔案會依照衛福部公告，內容以 EEC 格式為主。

業務端對應指南

Sales Guide

如何回覆醫院客戶？

Q: 什麼時候能配合一站通？
模組預計 3 月中開發完成，配合訂單流程建議報予客戶 2 個月內上線。
Q: 我們是 1.0 版可以裝嗎？
可以，不論是 EMR 1.0 或 2.0，我們均有提供專屬模組來支援 XML 匯出。

業務端行動清單

  • 確認客戶 EMR 版本 (1.0 或 2.0)。
  • 索取一站通測試帳號與 XML 存放目錄。
  • 定義多院區客戶之需求範圍。
